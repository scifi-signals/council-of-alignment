What is OpenRouter?

OpenRouter lets you access multiple AI models (Claude, GPT, Gemini, Grok) with one account and one API key. Sign up is free — you only pay for what you use, with prepaid credits so there are no surprise bills.

After signing up, go to Keys in your OpenRouter dashboard to create an API key, then paste it above.

Key Security

  • Encrypted at rest — your key is encrypted with AES-256 before it touches the database. We never store it in plaintext.
  • HTTPS only — all traffic between your browser and our server is encrypted in transit.
  • Used only for your reviews — your key is decrypted only when you convene a Council review, then discarded from memory.
  • Delete anytime — remove your key instantly from Settings. Once deleted, the encrypted data is gone.
  • Access audited — every key operation is logged so we can detect unauthorized access.

Tip: You can create a dedicated OpenRouter key with a spending limit just for the Council, and revoke it anytime from your OpenRouter dashboard.
